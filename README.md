# meta-legal-pages

Sitio estático para GitHub Pages de la app "Contenido Noticias Sobremesa".

## Páginas incluidas

- `index.html`
- `privacy.html`
- `terms.html`
- `data-deletion.html`

## Publicación en GitHub Pages

1. Crea un repositorio llamado `meta-legal-pages`.
2. Sube el contenido de esta carpeta a la rama principal.
3. En GitHub, activa Pages desde `Settings > Pages`.
4. Selecciona la rama `main` y la carpeta raíz `/`.
